# FILE HANDELING

### _1. File Handeling Life Cycle_
    1. Open the file
    2. Perform operation (read/write/append)
    3. Close the file
### _2. ALl Modes in which you can access the file_
| Mode | Meaning                       |
| ---- | ----------------------------- |
| `r`  | Read (default)                |
| `w`  | Write (creates or overwrites) |
| `a`  | Append                        |
| `x`  | Create new file               |
| `r+` | Read + Write                  |
| `w+` | Write + Read                  |
| `a+` | Append + Read                 |
| `b`  | Binary mode                   |
| `t`  | Text mode (default)           |

### _3. Opening a File in a Specific Mode_
    syntax -> file_object = open("filename", "mode")
    note -> mltiple modes can be combined, except the main modes
            | main modes  |
            | r -> read   |
            | w -> write  |
            | a -> append |
            | x -> create |
