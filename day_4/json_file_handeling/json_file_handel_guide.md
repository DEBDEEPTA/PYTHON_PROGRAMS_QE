# Handeling Json Files

### _1._
| JSON Type | Python Type |
| --------- | ----------- |
| object    | dict        |
| array     | list        |
| string    | str         |
| number    | int / float |
| boolean   | bool        |
| null      | None        |

### _2._

| Methods      | Descriptions           |
|--------------|------------------------|
| json.dump()  | write JSON to File     |
| json.dumps() | Convert to JSON String |
| json.load()  | Read JSON from file    |
| json.loads   | Parse JSON String      |

### _3._

            ↓ dumps()
    JSON string
